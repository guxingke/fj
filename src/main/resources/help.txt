Example usage:
  fj init
  fj info scaffold // coming soon...
  fj install(i) scaffold // install scaffold repo to fj
  fj uninstall(ui) scaffold // reverse for install
  fj list(ls) [ALIAS] // show scaffold list for given scaffold repo, default for all repo
  fj new(n) scaffold [target] // core action, generate scaffold to target dir
  fj status(st) // like git status, dir sensitive.

Troubleshooting:
  fj config // show cfg, dir sensitive.

Further help:
  fj cmd -h
